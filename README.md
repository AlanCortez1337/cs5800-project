# Bean Stock (Inventory Management App)

Simple application to add ingredient stock, create template recipes from ingredients, see up-to-date analytics based on ingredients & recipes

## Tech Stack & Libraries
### Frontend
- NextJS
- Shadcn
- Tailwindcss
- TanStack Query
### Backend
- Springboot
- PostgresSQL
